# Тенісна ліга

Красивий статичний сайт для жіночої тенісної ліги: рейтинг, матриця матчів, останні результати й live-оновлення з Google Sheet CSV.

## Як підключити live-дані

1. Веди матчі в Google Sheet, аркуш `Матчі`.
2. GPT/Codex може додавати нові матчі в цей аркуш за твоїми повідомленнями.
3. У Google Sheet обери `File -> Share -> Publish to web`.
4. Опублікуй аркуш `Матчі` як CSV або відкрий доступ `Anyone with the link -> Viewer`.
5. Встав CSV URL у `config.js`.

Зараз `csvUrl` уже підключений до аркуша `Матчі` в Google Sheet `Tennis League LIVE`.
Якщо Sheet не відкритий для перегляду за посиланням або не published, браузер друзів не зможе прочитати CSV.

Сайт підтримує два формати аркуша матчів:

```text
Матч
Бородай - Омельяненко 6:3 6:4
```

або структурований формат:

```text
Дата додавання,Гравчиня 1,Гравчиня 2,Рахунок,Переможниця,Програвша
2026-06-12,Бородай Олександра,Омельяненко Анна,6:3 6:4,Бородай Олександра,Омельяненко Анна
```

Поки live CSV недоступний, сайт може показувати `data/screenshot-matches-review.csv` як fallback.

## Як підключити live-розклад

У `config.js` є окреме поле `scheduleCsvUrl`. Воно очікує published CSV з колонками:

```text
Дата,День,Час,Гравчиня 1,Гравчиня 2,Примітка
13.06.2026,субота,09:00,Стрижак Олена,Рубець Юлія,Матч туру
```

Поки `scheduleCsvUrl` порожній, сайт показує `data/schedule-june-2026.csv`, перенесений з афіші розкладу.

## Формат матчу

```text
Бородай - Омельяненко 6:3 6:4
Омельяненко - Бородай 3:6 4:6
Антощенко - Литвиненко 6:7 6:2 10:8
```
